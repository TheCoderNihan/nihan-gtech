border-style property specifies what kind of border to display (inline css)

border-color property specifies color of border to display (inline css)

A dotted border (using class selector)

A red dashed border

A green solid border

A double border

A groove border

A ridge border.

An inset border.

A blue outset border.

No border.

A hidden border.

A mixed border.

The border color property does not work if it is used alone. Use border-style property to set the borders first
