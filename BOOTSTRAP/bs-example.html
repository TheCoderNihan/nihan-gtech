Bootstrap is simply a large collection of pre-written CSS classes that is given in the link tag.


Bootstrap has predefined styles. If you want custom styles , write css files and include your custom classes also in the html files.
Bootstrap and css work together and both are used in most projects.


Here the classes btn and btn-primary are already defined in bootstrap and we dont have to give the css code for it.


The recommended practice is giving bootstrap link in head tag and the javascript link in the body tag.
