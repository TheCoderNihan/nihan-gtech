Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page. If you do not want to use all 12 columns individually, you can group the columns together to create wider columns. Make sure that the sum adds up to 12 or fewer (it is not required to use all 12 available columns).


Bootstrap's Grid System (row and col) is designed to work inside a container. Used to wrap content.

container gives responsive fixed width and container-fluid always gives 100% width.
