Bootstrap Notes

Bootstrap is built using HTML, CSS and JavaScript. It provides thousands of pre-written CSS classes that developers can directly use.

Bootstrap does not replace CSS.It reduces the amount of CSS you need to write, but custom CSS is still used in almost every real-world project.

Professional websites commonly use the combination of HTML + Bootstrap + Custom CSS + JavaScript.

Bootstrap works by adding predefined classes such as btn, container, row, card, and text-primary to normal HTML elements.

Bootstrap classes are written inside the class attribute without using a dot (.). Example: class="btn btn-primary".

When writing CSS, class selectors begin with a dot (.). Example: .btn or .myClass.

Bootstrap already provides common colors such as Primary, Secondary, Success, Danger, Warning, Info, Light, Dark, White and Black.

If Bootstrap does not provide the color or style you want, create your own CSS class in an external stylesheet and use it together with Bootstrap classes.

Bootstrap CSS is loaded inside the head section using the link tag.

Bootstrap JavaScript is usually loaded just before the closing /body tag using the script tag.

Bootstrap JavaScript is required for interactive components such as Modals, Dropdowns, Accordions, Carousels, Navbar Toggle and Dismissible Alerts.

bootstrap.bundle.min.js already includes Popper.js, so no separate Popper library is required in Bootstrap 5.

CDN (Content Delivery Network) is a network of servers that stores and delivers files quickly over the Internet.

Bootstrap 3 required the jQuery library or many components to function properly.

Bootstrap 5 does not require jQuery because it is written using modern JavaScript.

Bootstrap 5 is the recommended version for modern web development and new projects.

Components such as Jumbotron, Wells, and Glyphicons belong mainly to Bootstrap 3 and are usually studied today only for older projects or academic examinations.

The most commonly used Bootstrap components include Containers, Grid System, Buttons, Typography, Cards, Forms, Tables, Images, Alerts,Navbars, Modals, Carousels, Accordions and Utility Classes.
